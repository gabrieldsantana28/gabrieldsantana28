### 🚀 Data Analyst | Data Engineering Enthusiast | Software Engineering Student

I'm a 19-year-old Software Engineering student from Brazil who is passionate about transforming data into business value.

My journey in technology started in 2023 during my Technical Degree in Systems Development at SENAI. While I explored different areas of software development, I discovered that my true interest lies in Data Analytics and Data Engineering.

Today, I work in the healthcare sector, building and maintaining data solutions that support strategic decision-making. My daily activities involve ETL/ELT processes, data pipelines, Oracle databases, Python automation, SQL development, and Power BI dashboards.

I'm constantly improving my skills in data architecture, analytics, and engineering, aiming to become a Data Engineer and work on large-scale data platforms in the future.
